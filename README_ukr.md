# Zurui 🚀

[🇺🇸 English](README.md) | [🇯🇵 日本語](README_jap.md) | [🇺🇦 Українська](README_ukr.md)

**Zurui** - це високошвидкісна комунікаційна платформа на ранній стадії, з нульовим збором даних, без роздування, без штучного інтелекту та рівня налаштування Arch Linux.

----------------------

### Індивідуальна реалізація QUIC

- [zclp++ 🚀](https://github.com/example/zclp++) - Реалізація C++ QUIC
- [zclp-rs 🦀](https://github.com/example/zclp-rs) - Реалізація Rust QUIC
